# Slides sugeridas

## 1. Portada

- Nombre: `EcoSort`
- Eslogan: `Tu residuo, en el lugar correcto.`
- Integrantes del equipo
- Logo

## 2. El problema

- La mayoria de errores ocurre cuando el usuario no sabe en que contenedor va el residuo.
- Ejemplos concretos: botella sucia, carton con grasa, pilas usadas.

## 3. Nuestra solucion

- Foto + texto corto
- Respuesta inmediata
- Clasificacion + reciclabilidad + guia

## 4. Como funciona la app

- Captura de imagen
- Campo de texto
- Pantalla de resultados

## 5. Como funciona la IA por dentro

- CNN para imagen
- Encoder textual para contexto
- Fusión final

## 6. Dataset y metodologia

- 15,515 imagenes
- 12 clases
- Texto sintetico controlado
- Split train/val/test

## 7. Resultados

- Accuracy
- Macro F1
- Matriz de confusion
- Aprendizajes del entrenamiento

## 8. Impacto y mercado

- Usuarios: hogares, escuelas, campus, municipios
- Impacto: menos contaminacion en reciclaje
- Escalabilidad: educacion, smart cities, puntos verdes
